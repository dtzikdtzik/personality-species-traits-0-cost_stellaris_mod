# personality-species-traits-0-cost_stellaris_mod
Small species trait mod. Adds 8 personality-themed 0-cost traits for individualist organic empires.
Work like the 0-cost machine traits. You can only pick one, but it doesn't cost a trait pick.

Flavor text and icons included.

1. Conservative: -50% ethics shift speed, +10% governing ethics attraction
2. Progressive: +100% ethics shift speed, -5% governing ethics attraction
3. Melancholy: +10% Job Efficiency for Influential Jobs, -5% Happiness
4. Stoic: -5% Amenities usage
5. Ambitious: -5 years leader starting age
6. Impulsive: +10% leader experience, -5 years leader lifespan
7. Meticulous: +10% Job efficiency for enforcers
8. Integrity: +10% Job efficiency for politicians
